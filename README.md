# VNNLIB-Standard 

This repository contains the source for the standard document for VNN-LIB. 
See the official [VNNLIB website](https://www.vnnlib.org/) for more details, including tooling for interacting with the specification.

### Contents

This repository contains:
- `document/main.tex` - the standard document 
- `grammar.cf` - a formal grammar for VNNLIB queries written in Labelled Backus-Naur Form.
